eonx-com/easy-bank-files

Parsers/Generators for common bank files

Introduction
Last Updated: 21/10/2020 06:25:05 - Edit on github

This package is dedicated to bank files parsing and generating.


Require package (Composer)

The recommended way to install this package is to use Composer:

$ composer require eonx-com/easy-bank-files


It supports parsing file formats as following:

  • BPAY BRF
  • BAI
  • NAI
  • ACK

It supports generating file formats as following:

  • ABA
  • BPAY Batch
